# 💼 Personal Developer Portfolio

A modern, responsive personal portfolio built using **Next.js, TypeScript, and Tailwind CSS**.  
This portfolio showcases my skills, projects, experience, and contact information in a clean and professional UI.

---

## 🌐 Live Demo

👉 https://gobinathan-k--portfolio.vercel.app/

---

## ✨ Features

- ⚡ Fast and optimized with Next.js
- 🎨 Modern UI using Tailwind CSS
- 📱 Fully responsive (mobile + desktop)
- 🧑 About section with developer introduction
- 💼 Experience timeline section
- 🛠 Services section showcasing skills
- 📬 Contact section with social links
- 🚀 SEO-friendly structure

---

## 🛠 Tech Stack

- **Framework:** Next.js
- **Language:** TypeScript
- **Styling:** Tailwind CSS
- **Deployment:** Vercel

## 🚀 Getting Started

Follow these steps to run the project locally:

### 1. Clone the repository
--bash
git clone https://github.com/Gobinathank/portfolio.git

### 2. Navigate to project folder
  cd portfolio
### 3. Install dependencies
   npm install
### 4. Run development server
   npm run dev
Open:
👉 http://localhost:3000

📦 Build for Production
To create an optimized production build:
   npm run build

To start production server:
  npm start
